AgentsPay

Payment infrastructure for AI agents. Give your agents wallets with spending limits, approvals, and real-time transaction tracking.

For Developers

CLI + SDK to create agents, fund wallets, and track spend from your code or terminal.

For AI Agents

MCP server + SDK for agents to request wallets, make payments, and check balances autonomously.

Quick Start — CLI

npm install -g @usectl/cli

agentspay auth register --email you@example.com --password yourpassword
agentspay auth login    --email you@example.com --password yourpassword

# Create an agent and give it a wallet
agentspay agent create --name "my-agent"
agentspay wallet create --agent-id <AGENT_ID> --name "main-wallet"

# Fund the wallet and check balance
agentspay wallet fund   --id <WALLET_ID> --amount 50.00
agentspay wallet get    --id <WALLET_ID>

# View transactions
agentspay tx list

Quick Start — SDK

npm install @usectl/sdk
import { AgentsPay } from '@usectl/sdk';

const ap = new AgentsPay({ apiKey: 'your-api-key' });

// Create an agent
const agent = await ap.agents.create({ name: 'my-agent' });

// Create and fund a wallet
const wallet = await ap.wallets.create({
  agent_id: agent.id,
  name: 'main-wallet',
  funding_amount: 100.00,
  limit_per_tx: 25.00,
});

// Check balance
const balance = await ap.wallets.getBalance(wallet.id);

MCP Server (for Claude & AI agents)

npm install -g @usectl/cli

Add to your MCP config (claude_desktop_config.json or similar):

{
  "mcpServers": {
    "agentspay": {
      "command": "npx",
      "args": ["-y", "@usectl/mcp"],
      "env": {
        "AGENTSPAY_API_KEY": "your-api-key"
      }
    }
  }
}

Available tools: create_wallet, get_balance, submit_payment, list_transactions, register_merchant, check_merchant, get_spending_summary
